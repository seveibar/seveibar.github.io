Visual CAD
An effort to combine OpenJSCad and Google Blockly to create a Computer Aided Design (CAD) tool for children to easily create complex 3D designs suitable for 3D printing.

Check it out on github!
Homework Server
A Rensselaer Center for Open Source project to build a better homework server for the RPI computer science department. The project involves improving algorithms used for grading homework, rewriting the backend php used for uploading and processing assignments, modernizing the webpages and making the assignment creation process more friendly to instructors.

Check it out on github!
Gunderbar
Gunderbar is an online service that offers deep integration and a seamless interface for sharing items with friends and finding items to borrow. It was first proposed at the 2013 RPI Business Model Competition and has been under development since.

Ultimate Miner
An Android game developed indendently using Adobe Air. Ultimate Miner 1 has been released with 100,000+ downloads and is available from the Google Play Store. Ultimate Miner 2 is currently under development.
